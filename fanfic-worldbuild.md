## 目标

**世界观构建与设定扩展。** 为同人或原创小说构建自洽的世界观，包括势力关系、魔法/科技体系、地理、种族、历史等。输出可直接作为写作参考文档。

**适用类型：** 赛博朋克、群星、西幻、科幻、轻小说、原创世界观。

## 输入

- 基础世界观（游戏/作品名称，或"原创"）
- 作者已有的二创/原创设定（如有）
- 需要扩展的方向（势力、魔法体系、科技树、地理、种族、历史事件）

## 流程

1. **原作校验。** 先确认原作中相关设定的canon内容，必要时搜索wiki。
2. **缝隙发现。** 找到原作没有明确定义但合理存在的空白区域。
3. **扩展构建。** 在原作框架内填充细节，确保不与canon冲突。
4. **内部一致性检查。** 扩展设定之间、扩展与canon之间不能矛盾。

## 输出格式

```markdown
## 设定名称

### Canon基础
原作中已确定的相关设定，引用来源。

### 扩展设定
作者可用的新增设定，标注与canon的关系（填充空白/合理推演/纯原创）。

### 使用建议
这个设定可以怎么用在剧情里，给出2-3个场景钩子。

### 冲突风险
可能与canon或已有二创设定冲突的地方，标注风险等级。
```

## 约束

- 扩展设定不能否定canon，只能补充。
- 标注每条设定的"硬度"：硬设定（不可改）vs 软设定（可调整）。
- 如果涉及多个世界观的crossover，明确标注哪些设定来自哪个世界观。
