# Antigravity adapter

Render `commands/*.md` as Antigravity workflows.

Current verified local target:

- `~/.gemini/antigravity/global_workflows/*.md`

Native MCP and skill support is tracked in `registry/target-capabilities.json`. Local user wiring currently includes `agentmemory` in `~/.gemini/antigravity/settings.json` and optional external `SKILL.md` packs in `~/.gemini/antigravity/skills/`.

Official primitive documentation is weaker than local evidence, so writes should stay behind dry-run review until verified again.
