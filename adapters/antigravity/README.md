# Antigravity adapter

Render `commands/*.md` as legacy Antigravity desktop workflows.

Default user install target:

- `~/.gemini/antigravity/global_workflows/*.md`

Google announced that consumer Gemini CLI users should migrate to Antigravity CLI by June 18, 2026. The latest CLI plugin package is generated by the separate `antigravity-cli` target.

Native MCP and skill support is not currently generated; local legacy user wiring for optional services belongs in the host config.
