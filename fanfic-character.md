## 角色设计、声音定义与弧线校准

为同人或原创小说创建、分析和校准角色。重点不是堆空泛人设，而是把角色的声音、欲望、矛盾、行为逻辑和变化路径钉牢：他怎么说、怎么想、怎么行动，在压力和亲密里怎么变形。

## 系列导航

- 何时使用：你还分不清角色怎么开口、怎么撒谎、怎么决策，或者你发现多名角色开始串音。
- 上一步：`fanfic-outline`、`fanfic-worldbuild`都可以先行。
- 下一步：`fanfic-relationship`、`fanfic-write`。
- 切到`fanfic-literary`：如果角色主要靠内心纹理、不可靠叙述、意识流或沉默成立，而不是外显台词与动作。
- 逐章校准：如果角色已经写进正文，需要做逐章返修而不是单独建卡，回`fanfic-improve`。

## 套件共识

- **角色先是人，再是功能。** 主角、反派、爱人、导师这些标签不等于人物本身。
- **声音不是口头禅。** 真正的差异来自词汇、句法、回避方式、价值排序和压力反应。
- **矛盾比标签重要。** “善良但坚强”没用，具体到他在关键抉择里会怎么拧巴。
- **模式影响声音。** 网文常靠台词和动作，文学向也可以靠内在措辞、叙述失真和沉默。

## 输入

- 角色名称（已有角色或原创角色）
- 所属世界观
- 在故事中的定位（主角 / 配角 / 反派 / 导师 / 爱人 / 宿敌 / 其他）
- 模式（网文 / 严肃文学 / 混合）
- 可选：已有正文片段、代表性台词、关系对象、角色弧线需求

## 先抓四个核心钉

- **欲望：** 这个角色此刻最想要什么。
- **防御：** 他通常怎么保护自己，嘴硬、躲闪、攻击、讲笑话、装冷静、讲道理？
- **破口：** 什么人或什么事会让他的语言和行为失控？
- **矛盾：** 他一边想要什么，一边又怕什么？他最大的自我分裂在哪里？

没有这四项，声音和弧线都很难稳定。

## 可选方法工具箱

- **声音指纹法：** 常用词、句长、语法偏好、默认语气、禁语、绕弯方式。
- **压力态法：** 平静、撒谎、愤怒、羞耻、亲密、失败时怎么变。
- **矛盾核心法：** 角色嘴上相信什么，身体其实相信什么。
- **弧线节点法：** 起点、盲点、第一次裂缝、中段反噬、高潮选择、终点状态。
- **遮名可辨法：** 遮住人名后，能否仅凭台词和反应认出他。

## 分析维度

### 基础信息

姓名、身份、外貌辨识点、职业/位置、所属势力、社会位置。

### 声音指纹

- 常用词汇、口头禅、禁语
- 句子长度偏好（短促 / 中等 / 长句）
- 语法倾向（完整句 / 碎句 / 并列句 / 绕弯子 / 书面化）
- 是否使用俚语、术语、外语词、黑话、书面语
- 默认语气（讽刺 / 冷淡 / 热情 / 谨慎 / 命令式 / 装松弛）
- 爱说实词还是抽象词

### 压力态变化

- 撒谎时：更啰嗦、更短、更解释过头，还是异常镇定？
- 紧张时：语速、断句、动作、视线怎么变？
- 愤怒时：沉默、冷笑、爆粗、讲大道理，还是直接动手？
- 面对最在乎的人时：哪些防御会松掉，哪些反而会更重？

### 行为模式

- 面对威胁的第一反应（战 / 逃 / 谈 / 算计 / 装无所谓）
- 道德底线在哪，什么绝对不做
- 弱点是什么，谁能触发
- 日常习惯和小动作
- 决策方式（直觉 / 算账 / 道德 / 面子 / 习惯）

### 关系与弧线

- 他最需要谁，又最怕被谁看穿？
- 他的盲点是什么？
- 如果他要变化，第一道裂缝会在哪？
- 如果他不变化，代价是什么？

## 声音测试

至少写3段不同情境下的样本：

1. 日常闲聊
2. 高压 / 危险场景
3. 面对最在乎的人或最恨的人

然后做三项测试：

- **遮名可辨测试：** 遮住名字，还能看出是谁在说吗？
- **去口头禅测试：** 拿掉最显眼的表层标签后，还能辨认吗？
- **内外反差测试：** 这角色的内心和外在是否有稳定张力，而不是表里一模一样？

## 模式提醒

- **网文：** 声音差异优先体现在台词速度、嘴损方式、冲突中的决策。
- **严肃文学：** 声音也可以体现在内心措辞、叙述失真、沉默和回避方式，不只看台词。
- **混合：** 外显台词和内在纹理都要稳定，别一到内心戏就换了个人。

## 边界

- canon角色不能随意改核心性格，只能补细节、稳定输出、合理外推。
- 不写空洞标签，如“善良但坚强”“外冷内热”，而要写具体选择、语言习惯和反应方式。
- 声音差异不能只靠口音、脏话和口头禅。真正的区别在句法、词汇、回避方式和价值排序。
