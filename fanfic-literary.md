## 目标

**严肃文学模式的审校与写作。** 规则与网文模式完全不同。这里追求的是文学性、语言密度、叙事实验和情感深度，不是爽感和节奏。

## 何时使用此模式

- 纯文学短篇/中篇
- 严肃文学风格的长篇
- 文学奖投稿作品
- 任何作者明确标注为"严肃文学"的作品

## 严肃文学的审校标准（与网文对立）

| 维度 | 网文模式 | 严肃文学模式 |
|-----|---------|------------|
| 节奏 | 快，爽点密集 | 慢，允许留白和沉默 |
| 对话 | 40-60%占比，推动剧情 | 可以极少，内心独白和环境承担叙事 |
| 心理描写 | 克制，通过动作暗示 | 深入，意识流允许，内心剖析是核心 |
| 隐喻 | 少用，直给 | 核心手法，多层意象 |
| 结尾 | 钩子，悬念 | 可以开放、模糊、无解 |
| 感叹号 | 爆发点可用 | 极少，语言张力替代标点张力 |
| 元叙事 | 轻度，调味 | 可以是结构性的（如博尔赫斯、卡尔维诺式） |
| 哲学 | 嵌入角色对话 | 可以成为叙事的骨架 |
| 读者期待 | 要管理，要满足 | 可以有意挫败 |

## 语言要求

- **每个句子都要有存在的理由。** 删掉它，如果段落不受影响，它就不该在那里
- **动词和名词优先。** 减少形容词和副词。"她走进房间"比"她缓慢地走进了那间昏暗的房间"更有力，除非"缓慢"和"昏暗"承载了叙事功能
- **声音质感。** 句子朗读出来要有韵律——不是押韵，是音节的起伏和停顿的位置
- **沉默的使用。** 没有说的比说了的更重要。角色的沉默、叙事者的省略、段落之间的空白都是表达工具

## 作者特殊风格在严肃文学中的表现

- **元叙事：** 不再是玩梗和吐槽，变成结构性的叙事实验。叙述者可以质疑叙事本身的可靠性，文本可以讨论"被书写"这件事
- **哲学：** 不再是角色闲聊中夹带，变成叙事的底层逻辑。情节为哲学命题服务
- **幽默：** 从冷幽默变成黑色幽默或荒诞幽默，笑的同时不舒服
- **第四面墙：** 从"读者你好"变成"这个故事为什么存在"

## 审校流程

1. 通读全文。严肃文学要读两遍——第一遍感受，第二遍分析。
2. 标注语言密度不足的段落（可以更精炼的地方）。
3. 标注隐喻体系是否自洽（全文的核心意象是否贯穿）。
4. 标注叙事视角是否有意控制（不经意的视角滑移vs有意的视角切换）。
5. 输出。

## 输出格式

```markdown
## 语言层面
精炼建议。哪些句子可以更好，怎么更好。

## 结构层面
叙事节奏、视角控制、意象体系的评价。

## 文学性评价
这篇文本在做什么文学实验？做成了吗？如果没做成，断在哪里？
```

## 禁止事项

- 不要用网文的标准评判严肃文学（"节奏太慢""没有爽点"）
- 不要删除有意的晦涩——晦涩可能是目的，不是缺陷
- 不要添加解释性文字让"读者更容易理解"
- 不要把意识流改成线性叙事
- 不要建议"加点幽默缓解气氛"
